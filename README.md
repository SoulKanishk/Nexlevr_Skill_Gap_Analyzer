# Nexlevr — Skill Gap Analyzer

A single-page, interactive tool that helps people compare their current skill levels against what a target role typically expects — and instantly see where the gaps are.

## What it does

- **Pick a target role** from a dropdown of career paths.
- **Self-rate your skills** using sliders for each relevant competency.
- **See the expected level** for that role marked directly on each slider (the gold tick), so you can compare your rating against the benchmark at a glance.
- **Get an instant gap view** showing your current level vs. what the role typically expects, helping you prioritize what to develop next.

## Live Demo

Since this is a static HTML file, you can host it for free with GitHub Pages:

1. Go to **Settings → Pages** in this repository
2. Under "Build and deployment," set Source to **Deploy from a branch**, branch = `main`, folder = `/ (root)`
3. Visit `https://<your-username>.github.io/<repo-name>/Nexlevr_Skill_Gap_Analyzer.html`

## Usage (locally)

No build step or dependencies required — it's a single self-contained HTML file.

1. Clone or download this repository
2. Open `Nexlevr_Skill_Gap_Analyzer.html` in any modern web browser

```bash
git clone https://github.com/<your-username>/<repo-name>.git
cd <repo-name>
open Nexlevr_Skill_Gap_Analyzer.html   # macOS
# or just double-click the file
```
