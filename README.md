# Nexlevr 🚀 — Skill Gap Analyzer

Nexlevr is a lightweight, single-page tool built around a simple idea: **what if figuring out your skill gaps took 30 seconds, not a career coach?** Pick a target role, rate yourself, and instantly see how you stack up — no signup, no backend, no friction.

Shipped as a single static HTML file so it's fast to run, fast to fork, and easy to build on. Feedback, forks, and PRs welcome.

## The idea

Most people *think* they know where their skill gaps are. Nexlevr makes it visual and immediate:

- 🎯 **Pick a target role** from a growing list of career paths
- 🎚️ **Self-rate your skills** with quick interactive sliders
- 📍 **See the benchmark** for that role marked right on the slider (the gold tick) — instant self vs. expected comparison
- ⚡ **Get your gap view** on the spot, so you know exactly what to work on next

No accounts. No data collection. No waiting. Just open it and go.

## Try it live

It's a static HTML file, so spinning up a demo costs nothing — GitHub Pages handles it:

1. Go to **Settings → Pages** in this repo
2. Under "Build and deployment," set Source to **Deploy from a branch**, branch = `main`, folder = `/ (root)`
3. Visit `https://<your-username>.github.io/<repo-name>/Nexlevr_Skill_Gap_Analyzer.html`

## Run it locally

Zero build step, zero dependencies — because prototypes should be fast to poke at.

1. Clone or download this repo
2. Open `Nexlevr_Skill_Gap_Analyzer.html` in any modern browser

```bash
git clone https://github.com/<your-username>/<repo-name>.git
cd <repo-name>
open Nexlevr_Skill_Gap_Analyzer.html   # macOS
# or just double-click the file
```
